Figure 1. Workflow diagram of the 3D genome analysis agent. The agent is constructed based on LangChain and LangGraph frameworks, integrating multiple specialized tools to implement end-to-end automated analysis of mcool files. The workflow consists of four main stages: knowledge retrieval, data processing, AI analysis, and result preservation. Blue nodes represent data retrieval and storage operations, while purple nodes represent AI analysis and processing operations. Diamond-shaped nodes indicate process branching and merging points.

使用说明 (Usage Notes):
• SVG 格式: 矢量图，适合在论文中使用，可以无限放大不失真，推荐用于期刊投稿
• PNG 格式: 高清位图（300 DPI），适合直接插入文档和演示文稿
• PDF 格式: 适合 LaTeX 文档使用，保持矢量特性
