            flowchart TD
                Start([开始]):::startend

                Start --> Init[初始化智能体]:::init

                Init --> Split1(( )):::split

                Split1 --> Knowledge[知识库检索
文献搜索]:::blue
                Split1 --> History[记忆库检索
历史记录]:::blue

                Knowledge --> Merge1(( )):::split
                History --> Merge1

                Merge1 --> Parse[需求解析

                Parse --> Determine[确定工具调用
选择合适的工具]:::purple

                Determine --> Split2(( )):::split

                Split2 --> ReadMcool[读取 mcool 文件
获取数据信息]:::green
                Split2 --> TADAnalysis[TAD 分析
识别拓扑结构域]:::green
                Split2 --> CompAnalysis[Compartment 分析
识别 A/B 区室]:::green

                ReadMcool --> Merge2(( )):::split
                TADAnalysis --> Merge2
                CompAnalysis --> Merge2

                Merge2 --> Integrate[结果整合
汇总分析结果]:::purple

                Integrate --> Verify[结论验证
文献知识库验证]:::purple

                Verify --> Save[保存结论
存储到数据库]:::blue

                Save --> End([结束]):::startend
