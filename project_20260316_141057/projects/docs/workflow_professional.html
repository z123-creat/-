Figure 1. Workflow of the 3D genome analysis agent. The system is built upon LangChain and LangGraph frameworks, orchestrating 10 specialized tools to achieve end-to-end automated analysis of mcool files. The workflow comprises four principal stages: (i) knowledge retrieval from literature database and historical records, (ii) requirement parsing and tool selection, (iii) parallel execution of Compartment analysis, TAD analysis, and general data processing, and (iv) result synthesis, validation, and persistent storage. Blue nodes represent data retrieval and storage operations, purple nodes indicate AI-driven analytical tasks, and diamond-shaped nodes denote control flow branching and synchronization points.

📊 Technical Specifications

  • Framework: LangChain 1.0 + LangGraph
  • Language: Python 3.12
  • Tools: 10 specialized instruments
  • Data Format: mcool (Hi-C contact maps)
  • Storage: Supabase PostgreSQL
