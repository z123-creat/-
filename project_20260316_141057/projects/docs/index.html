                    Decision --> |TAD分析| TADTools[TAD分析工具]
                    Decision --> |Compartment分析| CompTools[Compartment分析工具]
                    Decision --> |文件读取| ReadTools[数据读取工具]
                    Decision --> |历史查询| HistoryTools[历史记录工具]

                    TADTools --> TADProcess[1. 读取矩阵
2. 计算绝缘分数
3. 识别边界
4. 计算统计]
                    CompTools --> CompProcess[1. 计算OE矩阵
2. 执行PCA
3. 分配A/B
4. 计算统计]
                    ReadTools --> ReadProcess[1. 列出分辨率
2. 获取矩阵
3. 返回统计]
                    HistoryTools --> HistProcess[1. 保存结果
2. 查询历史]

                    TADProcess --> Knowledge[文献知识库]
                    CompProcess --> Knowledge
                    ReadProcess --> Knowledge

                    Knowledge --> Verify[结论验证]

                    Verify --> Output[生成报告]

                    Output --> End([输出给用户])

🎯 典型分析场景

场景 1: TAD 分析

                sequenceDiagram
                    participant U as 用户
                    participant A as Agent
                    participant T as TAD工具
                    participant K as 知识库
                    participant D as 数据库

                    U->>A: 分析 chr1 的 TAD 结构
                    A->>A: 解析需求
                    A->>T: analyze_tads(chr1, 10000bp)
                    T->>T: 读取矩阵
                    T->>T: 计算绝缘分数
                    T->>T: 识别 TAD 边界
                    T-->>A: 返回 TAD 结果
                    A->>K: 搜索相关文献
                    K-->>A: 返回文献片段
                    A->>D: 保存分析结果
                    A-->>U: 生成综合报告

场景 2: Compartment 分析

                sequenceDiagram
                    participant U as 用户
                    participant A as Agent
                    participant C as Compartment工具
                    participant H as 历史记录
                    participant D as 数据库

                    U->>A: 分析 chr1 的 A/B compartments
                    A->>A: 解析需求
                    A->>C: analyze_compartments(chr1, 10000bp)
                    C->>C: 获取接触矩阵
                    C->>C: 计算 OE 矩阵
                    C->>C: 执行 PCA
                    C->>C: 分配 A/B compartments
                    C-->>A: 返回分析结果
                    A->>H: 查询历史分析
                    H-->>A: 返回历史记录
                    A->>A: 对比验证
                    A->>D: 保存结果
                    A-->>U: 生成可视化报告
