Figure 1. System workflow of the 3D genome analysis agent. The framework consists of two main phases: the retrieval phase and the analysis phase. During the retrieval phase, the agent retrieves relevant information from both the knowledge base and historical analysis records. The analysis phase executes multiple analysis tasks in parallel, including Compartment analysis, TAD analysis, and general data analysis. Results are synthesized, validated, and stored in the database for future reference. All nodes represent sequential or parallel operations as indicated by the flow arrows.
