## 🔍 故障排查

### 问题 1：访问根路径仍显示 401

**可能原因**：
1. 文件未创建在正确位置
2. 服务未重新部署
3. 文件路径配置错误

**排查步骤**：

#### 检查文件是否存在
在 Coze 代码编辑器中确认：
```
src/
├── main.py
└── templates/
    ├── landing.html
    └── chat_with_token.html
```

#### 检查 main.py 路由
在 `src/main.py` 中确认包含：
```python
@app.get("/")
async def get_landing_page():
    """返回引导页面"""
    ...
```

#### 重新部署
1. 点击"保存"或"提交"
2. 点击"重新部署"
3. 等待部署完成（通常 1-3 分钟）
4. 再次测试

---

### 问题 2：引导页面显示但跳转失败

**可能原因**：
- `/chat` 路径配置错误
- JavaScript 错误

**排查步骤**：

1. 直接访问 `https://gsn2v3kydv.coze.site/chat`
2. 如果聊天页面正常，说明路径配置正确
3. 检查浏览器控制台是否有 JavaScript 错误

---

### 问题 3：聊天页面显示但连接失败

**可能原因**：
- 服务配置了认证
- API Key 错误

**排查步骤**：

1. 测试 `/health` 端点：
   ```bash
   curl https://gsn2v3kydv.coze.site/health
   ```

2. 如果返回 401，说明需要 API Key：
   - 在聊天界面输入 API Key
   - 点击"连接"

3. 如果返回正常，说明不需要认证
