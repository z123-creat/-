现在，您可以直接访问部署域名来使用三维基因组分析智能体，无需额外的配置！

## 🚀 快速开始

### 步骤 1：访问您的智能体

在浏览器中输入您的部署域名：

```
https://gsn2v3kydv.coze.site
```

或

```
https://您的域名.coze.site
```

### 步骤 2：连接服务

页面会自动加载，您会看到：

```
🧬 三维基因组分析智能体
```

**API 地址会自动配置为当前域名**，无需手动输入！

### 步骤 3：配置 API Key（如需要）

如果服务需要认证：

1. 在"API Key"输入框中输入您的 API Key
2. 点击"显示/隐藏"按钮可以查看或隐藏 API Key
3. 点击"连接"按钮

### 步骤 4：开始对话

连接成功后，您就可以开始与智能体对话了！

## 🔧 API Key 配置

### 获取 API Key

1. 登录 Coze 平台
2. 进入项目设置
3. 查找 API Key 或 Token
4. 复制 API Key

### 配置 API Key

1. 在聊天界面找到"API Key"输入框
2. 粘贴您的 API Key
3. 点击"连接"按钮

### API Key 说明

- **可选配置**：如果您的服务不需要认证，可以不填写
- **安全提示**：API Key 保存在浏览器的 localStorage 中，不会发送到第三方
- **隐私保护**：API Key 仅用于与服务端通信

## 🛠️ 故障排除

### 问题 1：页面无法加载

**可能原因**：
- 服务未启动
- 网络连接问题
- 域名解析失败

**解决方案**：
1. 检查服务是否正在运行
2. 测试网络连接
3. 确认域名是否正确
4. 尝试访问 `/health` 接口测试服务状态

### 问题 2：连接失败

**错误信息**：
```
连接失败：HTTP 401 Unauthorized
```

**解决方案**：
1. 检查 API Key 是否正确
2. 确认服务是否需要认证
3. 验证 API Key 是否已过期

### 问题 3：发送消息无响应

**可能原因**：
- 未连接到服务
- 网络中断
- 服务超时

**解决方案**：
1. 检查连接状态
2. 点击"测试连接"按钮
3. 刷新页面重试

### 问题 4：API Key 丢失

**现象**：
- 刷新页面后 API Key 消失

**原因**：
- 浏览器清除了 localStorage
- 使用了隐私模式

**解决方案**：
1. 重新输入 API Key
2. 避免清除浏览器数据
3. 使用正常浏览模式

## 📊 技术说明

### 架构设计

```
用户浏览器
    ↓
访问 https://域名.coze.site
    ↓
返回聊天界面 HTML
    ↓
JavaScript 自动检测域名
    ↓
配置 API 地址
    ↓
用户连接服务
    ↓
调用 /run 接口
    ↓
返回智能体响应
```

### 自动域名检测

页面加载时，JavaScript 自动获取当前域名：

```javascript
window.addEventListener('load', function() {
    const currentUrl = window.location.origin;
    document.getElementById('apiUrl').value = currentUrl;
});
```

### API 调用

前端调用后端 API：

```javascript
fetch(`${baseUrl}/run`, {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${apiKey}`
    },
    body: JSON.stringify({
        messages: [
            { role: 'user', content: message }
        ]
    })
})
```
